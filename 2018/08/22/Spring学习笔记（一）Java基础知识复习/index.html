5.==和equals的区别

  1. ==是判断两个变量或实例是不是指向同一个内存空间
    equals是判断两个变量或实例所指向的内存空间的值是不是相同

  2. ==是指对内存地址进行比较
    equals()是对字符串的内容进行比较

  3. ==指引用是否相同
    equals()指的是值是否相同

  4. 如果对象(都是继承了object(里面有equals方法))里重写equals了，又是不同结果
