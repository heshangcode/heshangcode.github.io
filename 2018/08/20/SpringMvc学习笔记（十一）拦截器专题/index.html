Java Web阶段学习过滤器

  • 过滤器是在服务器启动的时候就对齐进行实例化操作,如果你的过滤器错误,那么服务将无法启动,所以整个程序无法运行

    • doFilter(ServletRequest req , ServletResponse resp, ChainFilter chain)
      • 强制类型转换 HttpServletRequest request = (HttpServletRequest)req;

        • 访问下一个过滤器或者资源
    • destroy 服务器关闭
